# Business Card Website

A clean, single-page “business card” website: compact layout, readable typography, and a focus on content hierarchy.

## Highlights

- **Responsive layout** using modern CSS (flex/grid)
- **Clean typography** with local font files (Proxima Nova / SF)
- **Structured content blocks** (name, role, contacts, short bio)
- **Static project**: no build tools, no dependencies

## Project structure

```

  index.html
  css/
    style.css
  images/
    *.png
  *.otf
```
